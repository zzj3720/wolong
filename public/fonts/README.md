# 字体目录

此目录包含 LXGW WenKai（霞鹜文楷）字体文件。

## 必需的字体文件

需要以下字体文件：

- `LXGWWenKai-Regular.ttf` (font-weight: 400)
- `LXGWWenKai-Medium.ttf` (font-weight: 500, 600, 700)

注意：字体包仅提供 Regular 和 Medium 变体。Medium 用于 font-weight 500、600 和 700。

## 下载字体

### 方式一：自动下载（推荐）

运行下载脚本：

```bash
npm run download:fonts
```

或

```bash
pnpm download:fonts
```

### 方式二：手动下载

1. 访问 GitHub 发布页面：https://github.com/lxgw/LxgwWenKai/releases/latest
2. 下载以下字体文件：
   - `LXGWWenKai-Regular.ttf`
   - `LXGWWenKai-Medium.ttf`
   - `LXGWWenKai-SemiBold.ttf`
   - `LXGWWenKai-Bold.ttf`
3. 将所有下载的 `.ttf` 文件放置在此目录（`public/fonts/`）中
