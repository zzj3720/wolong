# Fonts Directory

This directory contains the LXGW WenKai (霞鹜文楷) font files.

## Required Font Files

The following font files are required:

- `LXGWWenKai-Regular.ttf` (font-weight: 400)
- `LXGWWenKai-Medium.ttf` (font-weight: 500, 600, 700)

Note: The font package only provides Regular and Medium variants. Medium is used for font-weight 500, 600, and 700.

## Download Fonts

### Option 1: Automatic Download (Recommended)

Run the download script:

```bash
npm run download:fonts
```

or

```bash
pnpm download:fonts
```

### Option 2: Manual Download

1. Visit the GitHub releases page: https://github.com/lxgw/LxgwWenKai/releases/latest
2. Download the following font files:
   - `LXGWWenKai-Regular.ttf`
   - `LXGWWenKai-Medium.ttf`
   - `LXGWWenKai-SemiBold.ttf`
   - `LXGWWenKai-Bold.ttf`
3. Place all downloaded `.ttf` files in this directory (`public/fonts/`)
